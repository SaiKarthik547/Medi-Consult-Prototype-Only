MediConsult is a production-grade, enterprise telemedicine backend architected to handle concurrent patient–doctor consultations with zero race conditions using SQL-level Compare-And-Swap. The system features a fully stateful consultation lifecycle (PAYMENT_PENDING → PAID → READY → ACTIVE → COMPLETED), real-time communication via WebSockets with Redis Pub/Sub fan-out, reliable side-effect delivery through a Transactional Outbox pattern, and HIPAA-aligned audit logging enforced at the ORM layer. Fault resilience is validated through five Toxiproxy chaos profiles including SIGKILL recovery and latency injection, ensuring correctness under real-world adverse conditions.

Key Engineering Highlights

⚡

Zero Race Conditions

SQL-level Compare-And-Swap with SKIP LOCKED — no distributed locks, no phantom assignments. Atomic patient pickup under heavy concurrency.

🧪

Chaos-Proven Resilience

5 test profiles using Toxiproxy fault injection — latency, packet loss, connection resets, timeout simulation, and SIGKILL recovery.

📦

Transactional Outbox

Side effects (emails, webhooks, notifications) committed atomically in the same DB transaction — guaranteed delivery without dual-write risk.

🔒

HIPAA-Aligned Audit

ORM-level immutability enforced on AuditLog — update/delete blocked at the library layer. Full actor/resource/timestamp trail for every event.
